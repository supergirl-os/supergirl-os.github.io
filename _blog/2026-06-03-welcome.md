This is a starter post for the new blog section.

In the future, notes can be added by creating new Markdown files in the `_blog`
folder. Each note can use the same front matter pattern as this file:

```yaml
---
layout: blog_post
title: "Post Title"
date: 2026-06-03
permalink: /blog/post-title/
excerpt: "A short summary for the Blog page."
---
```

Then write the post content below the front matter.
